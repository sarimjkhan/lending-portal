# Frontend: React Loan Calculator

## Overview
This React application interacts with the Django backend to create and manage loan offers and calculate monthly loan repayments.

## Setup Instructions

### Requirements
- Node.js 20
- npm or yarn

### Installation
1. Clone the repository.
2. Go to the project's root. 
3. Install dependencies:
   ```bash
   npm install

## Running the Application
### Start the application using:
```bash
npm start

## Running Tests
### Run tests with:
npm run test
